# /me greetings

Playful greetings for the home page — one is picked at random on each load.

Format: only lines starting with `-` are used. Use `{name}` for the user's first
name. Anything else (headings, blank lines, this note) is ignored, so organize
freely. Add/remove/reword at will.

- Look who's back.

- Can't get enough, can you.
- Couldn't resist, huh.

- Back in stock.
- So we're back.
- Here again?
- Right on schedule.
- And we're back.
- Back to business?
- Round two (or three, or…)
- Same place, same you.
- Well, well, well.
- Back so soon.
- I had a feeling.
- Took you long enough.
- There's the regular.
